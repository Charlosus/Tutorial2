Funkcja normalizeInput(input, to) deklaruje dwa parametry: input - ciąg znaków, który powinien mieć określoną wielkość liter to - ciąg z dwiema możliwymi wartościami: "upper" lub "lower", wskazującymi, jakimi literami powinna być napisana wartość input Uzupełnij kod funkcji tak, aby: Jeśli wartością parametru to jest ciąg upper, funkcja zwracała kopię ciągu input, ale zapisaną wielkimi literami W przeciwnym razie zwracała kopię ciągu input małymi literami Zadeklarowano funkcję normalizeInput(input, to) Wywołanie funkcji normalizeInput("This ISN'T SpaM", "lower") zwraca "this isn't spam" Wywołanie funkcji normalizeInput("This ISN'T SpaM", "upper") zwraca "THIS ISN'T SPAM" Wywołanie funkcji normalizeInput("Big SALE", "lower") zwraca "big sale" Wywołanie funkcji normalizeInput("Big SALE", "upper") zwraca "BIG SALE" Wywołanie funkcji normalizeInput("Stay Awhile and Listen", "lower") zwraca "stay awhile and listen" Wywołanie funkcji normalizeInput("Stay Awhile and Listen", "upper") zwraca "STAY AWHILE AND LISTEN"

Funkcja checkFileExtension(fileName, ext) przyjmuje dwa parametry: fileName - ciąg znaków przechowujący nazwę pliku z rozszerzeniem, na przykład styles.css, hello.js itp. ext - ciąg znaków przechowujący rozszerzenie, na przykład .css, .js itp. Uzupełnij kod funkcji tak, aby: Jeśli nazwa pliku fileName kończy się rozszerzeniem podanym w parametrze ext, funkcja zwracała ciąg znaków "File extension matches" W przeciwnym razie zwracała ciąg znaków "File extension does not match" Zadeklarowano funkcję checkFileExtension(fullname, name) Wywołanie funkcji checkFileExtension("styles.css", ".css") zwraca "File extension matches" Wywołanie funkcji checkFileExtension("styles.css", ".js") zwraca "File extension does not match" Wywołanie funkcji checkFileExtension("app.js", ".js") zwraca "File extension matches" Wywołanie funkcji checkFileExtension("app.js", ".html") zwraca "File extension does not match" Wywołanie funkcji checkFileExtension("index.html", ".html") zwraca "File extension matches" Wywołanie funkcji checkFileExtension("index.html", ".css") zwraca "File extension does not match" Wywołanie funkcji checkFileExtension("index.html", ".js") zwraca "File extension does not match"
